Linux使用过程中遇到的坑

  1. 更改密码时注意小键盘，ubuntu里面就算小键盘没开也是默认又输入的

  2. 系统莫名其妙在登陆界面卡住，鼠标键盘都不好使

    • 有可能是升级内核后造成的，利用如下命令就好了sudo apt-get install --reinstall xserver-xorg-input-all

  3. ubuntu下不能挂载Windows盘符的情况

    首先终端输入sudo fdisk -l 查看分区挂载情况

    然后修复相应的挂载错误区，输入sudo ntfsfix /dev/sda5
